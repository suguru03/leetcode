## 226. Invert Binary Tree

https://leetcode.com/problems/invert-binary-tree/description/

Invert a binary tree.

```
     4
   /   \
  2     7
 / \   / \
1   3 6   9
```

to

```
     4
   /   \
  7     2
 / \   / \
9   6 3   1
```
